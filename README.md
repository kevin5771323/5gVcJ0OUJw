# 电商后台管理系统 java1091

## 项目概述

本项目是基于Vue和Node.js的电商后台管理系统，实现了前后端分离的设计模式。系统包含了源码和数据库文件，致力于提供一套完善的用户、权限、商品及订单管理解决方案。

## 技术栈

### 后端技术
- Node.js

### 前端技术
- Vue2

### 开发环境
- 操作系统：Windows 10，11
- 开发工具：Visual Studio Code
- 编程语言：JavaScript
- 数据库：MySQL 8.2.0
- Node.js版本：v18.17.1

## 功能模块

- 登录
- 用户管理（增删查改）
- 权限管理（增删改）
- 商品管理（增删查改）
- 订单管理
- 数据统计（图标展示）

## 系统角色

- 系统管理员：负责用户、权限和数据统计的管理。
- 普通用户：进行商品和订单的相关操作。

## 运行环境

- 前端：现代浏览器
- 后端：Node.js运行环境

## 目录结构

```
project/
│
├── frontend/          # 前端项目目录
│   ├── src/
│   ├── public/
│   ├── ...
│
└── backend/           # 后端项目目录
    ├── routes/
    ├── models/
    ├── controllers/
    ├── ...
```

## 部署步骤

1. 准备好前后端开发环境。
2. 安装Node.js、MySQL等必备软件。
3. 克隆项目代码到本地。
4. 分别在前后端目录下执行安装依赖的命令。
5. 启动后端服务。
6. 启动前端服务。

## 核心亮点

- 前后端分离，便于团队协作和项目管理。
- 使用Node.js，确保后台逻辑处理的高效性。
- 集成权限管理，保障系统安全。
